Support

Support for the beta.

Home Floorplan is still evolving. These notes define the support areas that will be expanded as the beta opens.

Getting Started

Use an iPad in landscape, create or open a floorplan and start connecting rooms to your Apple Home context.

Floorplans

The floorplan is the main navigation model. Rooms, accessories and state should be placed where they make spatial sense.

Environment

Environment views are designed to make comfort and air quality readable by room rather than as disconnected sensor values.

Security

Security views use room context to make doors, windows and motion events easier to scan quickly.

Habits & AI

Habits should explain patterns and opportunities without becoming a generic chatbot experience.
